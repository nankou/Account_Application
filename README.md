# Account_Application
## 系统主题
本系统为个人记账app，基于JAVA实现的安卓应用。
根据之前使用的记账程序，功能繁多，但作为一个记账软件最重要的就是记账功能和查看账单的功能。于是本系统化繁为简，确定了以下几个功能点：
登录、注册、用户中心、查看账单、记录账单。

## 数据存储 
本系统采用SQLite数据库存储数据。SQLite是一款轻量级的关系型数据库，运算快，占用空间小。

## 运行
